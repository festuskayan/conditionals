Musical groups have special names based on the number of people in the group. For example, a "quartet" is a musical group with four musicians. Directions: Write a series of conditional statements that: Prints "not a group" if musicians is less than or equal to 0 Prints "solo" if musicians is equal to 1 Prints "duet" if musicians is equal to 2 Prints "trio" if musicians is equal to 3 Prints "quartet" if musicians is equal to 4 Prints "this is a large group" if musicians is greater than 4
